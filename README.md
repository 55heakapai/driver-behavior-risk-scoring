# Driver Behavior Risk Scoring

## Overview
This project analyzes driver behavior data and predicts risk levels using Machine Learning. It helps identify safe, moderate, and risky driving patterns based on driving metrics.

## Features
- Data preprocessing and cleaning  
- Exploratory Data Analysis (EDA)  
- Interactive visualizations in Google Colab  
- Risk score prediction using ML models  
- Driver behavior classification  

## Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Plotly  
- Scikit-learn  
- Google Colab  

## Project Workflow
1. Load dataset  
2. Clean and preprocess data  
3. Perform data analysis  
4. Train ML model  
5. Predict risk score  
6. Visualize results  

## Output
- Driver risk score  
- Behavior category (Safe / Moderate / Risky)  
- Interactive charts and insights
